# horribleSubsBot
A web scrapping bot that will download anime from HorribleSubs

More information coming soon...

## Dependencies
* python > 2.7
    * python packages:
        * Ghost.py == 0.2.3
        * lxml == 3.8.0
        * pymongo == 3.5.0
        * PySide == 1.2.4
        * regex == 2017.7.28
        * requests == 2.18.3
        * urllib3 == 1.22
        * webscraping == 1.6
* mongoDB > 3.2
